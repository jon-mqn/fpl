# FPL Multiverse

What if you hadn't made that trade? What if you stopped making trades at GW3? Each trade you make creates a new team. What if you could visualise the performance of all your teams from past gameweeks? Would my team from GW4 have beformed better in GW40 compared with my team from GW 36?

## Tools

### Python

- [fpl](https://github.com/amosbastian/fpl) - An asynchronous Python wrapper for the Fantasy Premier League API.
- [plotly](https://github.com/plotly/plotly.py) - The interactive graphing library for Python.
- [flask](https://github.com/pallets/flask) - The Python micro framework for building web applications.
